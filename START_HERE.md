# START HERE

Run this portable Acquisition Strategy Agent demo locally:

## macOS

Double-click `run-mac.command`, or run:

```bash
python3 server.py
```

## Windows

Double-click `run-windows.bat`, or run:

```powershell
python server.py
```

Then open:

```text
http://127.0.0.1:8123
```

For LAN sharing and troubleshooting, read `RUN_ON_OTHER_COMPUTER.md`.
